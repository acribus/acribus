# 主题配置

一个中后台项目的主题应该展现整个网站的布局和配色细节。在 arcibus 中，自带了三个主题（acribus、element、flatui），它是为各类使用场景而设计的。默认主题为acribus。

import 'acribus-theme/dist/acribus/index.css'

在 main.js 中修改对应的 css 文件引入即可更换主题，详情请参见 主题 。
