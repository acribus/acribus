          # 主题

          提示

          在大部分情况下，开发者无需了解本章内容，因为 acribus 可以自动生成符合主题规范的页面；但如果需要自行开发页面，或对主题有特殊的需求，则需要对本章节有一定的了解。

          # 介绍

          Acribus 使用了 acribus-theme 作为主题，这是基于 element-ui 封装的一套主题库，提供了若干配色方案和实用的 CSS 片段。

          # CSS 变量

          acribus-theme 使用了 CSS 变量来简化代码。由于 CSS 变量与 CSS 属性类似，也支持层叠，因此如果需要修改一些变量，只需要为 #app 添加对应的代码来覆盖原有的变量：

          #app {
  /* 正文的字体大小 */
  --font-size: 14px;
  /* header 的高度 */
  --header-height: 60px;
  /* 左侧边栏的宽度 */
  --left-side-width: 250px;
  /* 面包屑菜单的高度 */
  --breadcrumb-height: 20px;
  /* 卡片的圆角程度 */
  --card-border-radius: 5px;
  /* 卡片间的空白 */
  --card-spacing: 16px;
  /* 标签的圆角程度 */
  --tag-border-radius: 5px;
  /* 标签的字体大小 */
  --tag-font-size: 12px;
  /* 上传相关 */
  --uploader-size: 178px;
  --uploader-border-radius: 5px;
}

          # 配色方案

          acribus-theme 目前提供了三种配色方案：acribus、element、flat-ui。若要使用某一特定的配色方案，只需要在 main.js 中，将 element-ui 的 CSS 文件替换掉即可：

          - import 'element-ui/lib/theme-chalk/index.css'
+ import 'acribus-theme/dist/acribus/index.css'

          如果需要对主题的颜色进行一些配置，可以在 #app 中覆盖原有的颜色变量：

          #app {
  /* 基础配色 */
  --color-primary: #409eff;
  --color-success: #67c23a;
  --color-warning: #e6a23c;
  --color-danger: #f56c6c;
  --color-info: #909399;
  --color-disabled: #c0c4cc;

  /* 三级文字颜色 */
  --text-color-light: #fff;
  --text-color: #999;
  --text-color-dark: #333;

  /* 五级背景颜色 */
  --bg-color-lighter: #fff;
  --bg-color-light: #f0f2f5;
  --bg-color: #6b6b6b;
  --bg-color-dark: #5b5b5b;
  --bg-color-darker: #515151;

  /* 卡片的背景颜色 */
  --bg-card: #fff;
}

          # CSS 类名

          acribus-theme 提供的 CSS 类名均以 .sp- 开头：

          类名作用
          .sp-cursor-pointer鼠标指针变为 pointer，并禁止选中元素内的文字
          .sp-card声明一个卡片
          .sp-tag声明一个标签
          .sp-bg-{color}将背景颜色改为 {color}（例如 .sp-bg-primary）
          .sp-color-{color}将文字颜色改为 {color}（例如 .sp-color-primary）
