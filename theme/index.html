          # Theme

          TIP

          In most cases, developers don't need to understand the contents of this section, because acribus can automatically generate pages satisfied to our design specification; however, if you need to code for your own pages, or have special needs for themes, you need to have a certain understanding of this section.

          # Introduction

          Acribus uses acribus-theme as its theme, which is a theme library based on element-ui, providing several color schemes and useful CSS pieces.

          # CSS Variables

          acribus-theme uses CSS variables to simplify its code. Since CSS variables are similar to CSS properties, which supports cascading, if you need to modify some variables, only need to add some codes for #app to overwrite the original variables:

          #app {
  /* main font size */
  --font-size: 14px;
  /* header height */
  --header-height: 60px;
  /* left sidebar width */
  --left-side-width: 250px;
  /* breadcrumb height */
  --breadcrumb-height: 20px;
  /* border radius for card */
  --card-border-radius: 5px;
  /* space between cards */
  --card-spacing: 16px;
  /* border radius for tag */
  --tag-border-radius: 5px;
  /* font size for tag */
  --tag-font-size: 12px;
  /* these are for uploader */
  --uploader-size: 178px;
  --uploader-border-radius: 5px;
}

          # Color Scheme

          acribus-theme currently provides three color schemes: acribus, element, flat-ui. To use a specific color scheme, simply replace the CSS file for element-ui in main.js:

          - import 'element-ui/lib/theme-chalk/index.css'
+ import 'acribus-theme/dist/acribus/index.css'

          If you need to configure some colors of the theme, you can override the original color variables in #app:

          #app {
  /* basic colors */
  --color-primary: #409eff;
  --color-success: #67c23a;
  --color-warning: #e6a23c;
  --color-danger: #f56c6c;
  --color-info: #909399;
  --color-disabled: #c0c4cc;

  /* three-level text colors */
  --text-color-light: #fff;
  --text-color: #999;
  --text-color-dark: #333;

  /* five-level background colors */
  --bg-color-lighter: #fff;
  --bg-color-light: #f0f2f5;
  --bg-color: #6b6b6b;
  --bg-color-dark: #5b5b5b;
  --bg-color-darker: #515151;

  /* background color for card */
  --bg-card: #fff;
}

          # CSS Class Names

          The CSS class names which acribus-theme provides are all start with .sp- prefix:

          Class NameMeaning
          .sp-cursor-pointerChange cursor to pointer, and disallow selection for inner text
          .sp-cardDeclare a card
          .sp-tagDeclare a tag
          .sp-bg-{color}Change background to {color} (e.g. .sp-bg-primary)
          .sp-color-{color}Change text color to {color} (e.g. .sp-color-primary)
