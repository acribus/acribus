# Getting Started

COMPATIBILITY NOTE

Please ensure that your npm version or yarn version is correct

# Installation

you can use yarn or npm install arcibus

# 安装
yarn add acribus [--dev/-D] # 或者：npm install acribus -D

# Write configuration

It is very simple and quick to generate the corresponding list page, view page, create page and edit page by the following configuration.

# 引入
import {ListViewFactory} from 'acribus'

# 命名列表资源、每条资源的schema、每条资源的操作行为
ListViewFactory({
  data: new CollectionDescriptor('role'),
  schema: [
    new Schema.Input('Role ID', 'role_id', false, {
      editable: false,
      asFilter: true,
      render: (value, row) => row['id']
    }),
    new Schema.Input('Role Name', 'name', true),
    new Schema.Select('Status', 'status', [
      ...ROLE.STATUS.map((item, index) => ({
        label: item,
        value: index
      }))
    ], true, {
      asFilter: true,
      render: (value) => renderStatus({
        type: ROLE.STATUS_CLASS[value],
        content: ROLE.STATUS[value]
      })
    }),
    new Schema.Datetime('Create Time', 'ctime', false, {
      editable: false
    }),
    new Schema.Tree('Permission', 'permission_list', state => state.user.permissionTree, true, {
      editable: true,
      inTable: false,
      asFilter: false
    })
  ],
  actions: [
    'edit'
  ]
})

See Configuration to see all configurable options.
